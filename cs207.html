> welcome :)_

In this course, we will do an in-depth study of object-oriented programming (OOP) and abstract data structures implementation and application. With this knowledge, you will advance to design and build more sophisticated code, with greater structural complexity.

We will cover topics like inheritance and polymorphism, abstract classes and interfaces, exception handling, file operations, stacks, queues, lists, trees, heaps, and graphs. Additional topics include recursion, searching, and sorting algorithms.
